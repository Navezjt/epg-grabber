# EPG Grabber

Node.js CLI tool for grabbing EPG from different sites.

## Installation

```sh
npm install -g epg-grabber
```

## Usage

```sh
epg-grabber --config=example.com.config.js
```

Arguments:

- `-c, --config`: path to config file
- `-d, --debug`: enable debug mode
